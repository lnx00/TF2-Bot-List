Welcome!

This site was created to make the process of installing Bot Lists in Team Fortress 2 / TF2 Bot Detector easier.
You can simply download various file formats that will be converted automatically.
It is recommended to use Pazer's TF2 Bot Detector, that will kick bots from your game.

If you have problems installing / importing one of the lists then you can read the guides on the Download page.
They will also help you installing the TF2 Bot Detector.

Advanced users can also use Tools to convert your own Bot List into other formats (List, JSON, TF2 Bot Detector) or find bots that aren't added to wgetJane's Bot List yet.

This is an open source project than can be viewed here on GitHub.
