Guides / Help

  • filter_altHow to install the TF2 Bot Detector Playerlist
    1. Make sure that TF2 Bot Detector is installed correctly
    2. Download playerlist.tf2bl.json by clicking the corresponding button above
    3. Move playerlist.tf2bl.json into the cfg folder of TF2 Bot Detector
    4. Run TF2 Bot Detector and make sure the console says, that the file has been loaded
  • voice_over_offHow to install the Voice Ban file
    1. Download voice_ban.dt by clicking the corresponding button above
    2. Find and open your Team Fortress 2 game folder (Steam -> Properties -> Local Files -> Browse Local Files)
    3. Move voice_ban.dt into the tf folder located inside your Team Fortress 2 folder
  • get_appHow to install TF2 Bot Detector
    1. Download the latest TF2 Bot Detector by Pazer or view all releases
    2. Extract tf2_bot_detector_XXXX.zip into your desired folder (not your TF2 Game Folder)
    3. Run tf2_bot_detector.exe and follow the on-screen instructions
    4. The program should now look like this
    I don't have tf2_bot_detector.exe / What do I have to do with Visual Studio?

    If you don't have tf2_bot_detector.exe or Visual Studio is opening then you downloaded the wrong file.
    Make sure that you download tf2_bot_detector_XXXX.zip and nothing else (eg. symbols.zip)
    You can also download the latest Version automatically. Make sure to wait a moment if the download isn't starting immediately.

    The program says that VCRUNTIME140_1.dll was not found

    You are most probably missing Microsoft Visual C++ Redistributable for Visual Studio 2015, 2017 and 2019. You can download it here.
    Simply run the installer and follow the on-screen instructions.
